Caso 1: Contratar uma festa Logado

1- Usuário acessa o site
2- Acessa a aba de galeria
3- Seleciona a temática
4- Demonstra interesse
5- Redirecionado para o carrinho, aba da festa, onde pode editar os itens da festa
6- Envia a solicitação
7- Solicitação fica pendente para o gestor (Status: aprovado, pendente, negado)
8- Festa aprovada
9- Cliente contatado pelo whatsapp
10- Atualizado no carrinho (Status: Aprovado)
11- Gestor gera contrato para a pessoa
12- Cliente assina o contrato
13- A festa é realizada.

Caso 2: Gerenciar solicitações pelo Login de ADM

1- Acessa o Painel de Admin
2- Acessa solicitações
3 -Altera os Status da Festa (Padrão: no aguardo / Status: Aprovado, cancelado, negado).
